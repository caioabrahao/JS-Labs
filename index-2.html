A great place to Test Some Code.

Sometimes, I just want to sit in front of my computer, write some code, do some shenanigans in my IDE and make some funny stuff. For that, I use the JS Labs!

Testing is also Learning!

What if there was a way to turn my simple useless code into an interactive learning experience for new developers just like myself?!

But even tho it's not ready for the world just yet, we have a name already! We call it the JS Labs: Programming Education for New JavaScript Learners, or P.E.N.N.Y, for short.

As mentioned, it is still in the planning phase, but stay tuned for when it comes out!
